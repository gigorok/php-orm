PHP ORM library
=============================

## Installation

Use [composer](http://getcomposer.org) for installation PHP ORM library.
Just add to your `composer.json`
a text below and run the `php composer.phar update` command to install it:

    {
        "require": {
            "gigorok/php-orm": "0.1.*"
        }
    }
